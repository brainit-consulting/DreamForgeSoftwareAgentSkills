# Privacy Policy

**DreamForge Software — dreamforge-audit Plugin**
**Last updated:** March 24, 2026

## Overview

The `dreamforge-audit` plugin (audit-my-app) is an open-source skill that runs entirely locally within your Claude Code or Claude Cowork session. It does not collect, transmit, or store any personal data.

## What the plugin does

- Reads files in your local project directory to perform code audits
- Generates audit reports saved locally to your `AuditReports/` directory
- May suggest installing additional skills from skills.sh (with your explicit approval)
- May run local CLI tools (npm audit, TypeScript compiler, etc.) with your approval

## Data collection

**This plugin does not collect any data.** Specifically:

- No personal information is collected
- No usage analytics or telemetry is sent
- No data is transmitted to DreamForge Software or any third party
- No cookies or tracking mechanisms are used
- No API keys or credentials are stored by the plugin

## Data processing

All processing happens locally within your Claude Code or Claude Cowork session. The plugin:

- Reads your source code only during the audit session
- Does not persist any data beyond the generated audit report files
- Does not access any external services (unless you approve tool installations)

## Third-party services

The plugin may suggest installing skills from [skills.sh](https://skills.sh), which is a third-party service. Any such installation requires your explicit approval and is governed by that service's own privacy policy.
